# GoogleMaps-SwiggyClone
Project working as Intern, Implemented Google Maps SDK, both SwiftUI &amp; UIKit are used.

## Technology Used
- Language: Swift
- SwiftUI & UIKit(for Map)
- Google Maps, Google Places SDK, Core Location(Apple's Framework)
- MVVM Architecture
